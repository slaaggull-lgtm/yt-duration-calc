📖 HOW TO GET AN API KEY
  1. 01
    Go to Google Cloud Console and sign in with your Google account.
  2. 02
    Click "Create Project" → give it a name (e.g. "YT Calculator") → click Create.
  3. 03
    In the left sidebar go to APIs & Services → Library. Search for "YouTube Data API v3" and click Enable.
  4. 04
    Go to APIs & Services → Credentials → click "+ CREATE CREDENTIALS" → API Key.
  5. 05
    (Recommended) Click "Restrict Key" → under API restrictions select YouTube Data API v3 only.
  6. 06
    Copy your key and paste it in the API Key field above. Free quota is 10,000 units/day — more than enough.
